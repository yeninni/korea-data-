# Korea Easy Locker

대한민국을 방문한 외국인 관광객, 가족 관광객, 대형 짐 이용자를 위한 전국 관광거점 공공 물품보관함 다국어 안내 데모 웹사이트입니다.

## Concept

- 핵심 데이터 컨셉: 공영 물품보관함 현황 실시간 정보
- 확장 데이터 컨셉: 전국 초정밀 버스 실시간 위치 정보
- 지원 언어: 한국어, English, 中文, 日本語
- 주요 기능: 지역 필터, 관광거점 바로가기, 검색, 지도형 탐색, 실시간 상태 배지, 정렬, 대형 짐 필터, 버스 정보형 상세 카드
- 데모 커버리지: 서울, 부산, 인천, 제주, 경주, 강릉 등 외국인 방문이 많은 관광거점

## Run

```bash
npm install
LOCKER_API_KEY=your_public_data_key npm run dev
```

로컬 데모 주소는 기본적으로 `http://127.0.0.1:4173/` 입니다.
